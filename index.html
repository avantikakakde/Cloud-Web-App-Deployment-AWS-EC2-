🌩️ My Cloud App

Deployed on AWS EC2 — Powered by HTML, CSS, and JS

Welcome to My Cloud App!

This app is hosted on AWS EC2 — a fully customizable cloud server that gives you complete control.

It demonstrates a simple static website using HTML, CSS, and JavaScript.

App Features 🌟

  • ✅ Responsive design (works on mobile and desktop)
  • 🌗 Light/Dark mode toggle
  • 🧭 Tab navigation with smooth animations
  • 💻 Perfect for cloud hosting (AWS, GCP, Azure, GitHub Pages)
  • 🆓 100% static — no backend needed!

About This Project

This project is part of a Cloud Deployment Exercise — learning how to host simple apps on the cloud using EC2 instances.

Technologies used:

  • ☁️ AWS EC2 (Ubuntu + Apache)
  • 💻 HTML, CSS, JavaScript
  • 📦 GitHub for version control
